# 김우창 「세계와 문학의 세계」 분석

## 글 정보
- **저자**: 김우창
- **발표**: 1976년
- **성격**: 미학론, 문학 세계론

---

## 1. 인용 철학자/사상가/작가

| 인물 | 인용 맥락 |
|------|-----------|
| **에머슨(Ralph Waldo Emerson)** | 시 「낱과 모두(Each and All)」 - 아름다움은 사물 간의 관계에서 발생 |
| **스탕달(Stendhal)** | 자서전 『앙리 브륄라르의 생애』 서두 - 로마 자니쿨레 언덕의 풍경 묘사 |
| **플라톤(Plato)** | 『티마이오스』 - 지각 과정의 적극적 구성 작용 |
| **하이데거(Heidegger)** | 『예술 작품의 기원』 - 반 고흐의 구두 해석, 땅과 세계의 투쟁 |
| **미셸 푸코(Michel Foucault)** | "사물의 쌍둥이 같은 성질" - 비유의 구극적 근거 |
| **반 고흐(Van Gogh)** | 구두 그림 - 하이데거의 예술론 예시 |

## 2. 인용 예시문

### 2.1 에머슨 (Ralph Waldo Emerson) - 「낱과 모두(Each and All)」

> "들에 선 붉은옷의 광대는 / 언덕 위의 너를 생각지 않는다 / 윗 들녘에 우는 암소는 / 너의 귀를 위해 울지 않는다. / 알프스 험한 고개 넘는 나폴레옹 교회의 종소리 그를 위해 울지 않건만 울려오는 종소리 기쁘게 듣는다. / 너는 너의 삶이 이웃사람에게 어떤 교훈이 되는지 알지 못한다. 낱낱의 것은 모두를 필요로 한다. / 따로이어 아름답고 좋은 것은 없다. / 동틀무렵 참새 한마리 오리나무위에 하늘나라새인 듯 곱게 울기에 / 저녁 무렵 그새, 둥우리채 잡아왔다. / 새 노래는 같건만 내 마음은 기쁘지 않다. / 강과 하늘을 가져오지 않았기 때문에. / 새는 내 귀에, 강과 하늘은 내 눈에, / 함께 어울려 노래했던 것을 / 조개 하나 강가에 놓여 있어 / 밀려드는 물결 비취빛 조개에 진주를 매었다. / 조개를 집어 든 나에게 바다의 포효는 그 다행한 구조를 신호했다. / 해초와 물거품 씻어 낸 바다의 이 보석, 집으로 가져왔더니 이제 그것은 / 초라하고 흉하고 어지러운 물건, 아름다움 바닷가에 두고 온 듯 / 태양과 모래와 사나운 포효의 바닷가에……"

### 2.2 스탕달 (Stendhal) - 『앙리 브륄라르의 생애』

> "1832년 10월 16일 나는 오늘 아침 로마의 자니쿨레 언덕, 몬토리오의 생 피에트로에 갔다. 태양이 찬란한 날이었다. 가벼운 씨로코의 바람이 알바노의 산 위로 흰 조각구름을 떠가게 했다. 감미로운 열기(熱氣)가 공기를 채우고 있었다. 나는 살아 있다는 것이 행복했다. 40리 저쪽에 있는 프라스카티와 카스텔 곤돌프를 똑똑히 알아볼 수 있었다. 그리고 도미니캥 드 쥐디트의 아름다운 벽화가 있는 알도브란디의 별장……"

### 2.3 플라톤 (Plato) - 『티마이오스』

> "**플라톤**이 『티마이오스』에서, 우리가 사물을 볼 때 사람의 몸을 덮는 엷은 열기가 눈에서 빠져나가서 부드럽고 진한 빛의 흐름을 이룬다고 말한 것은 그렇게 틀린 관찰이 아니다."

### 2.4 하이데거 (Heidegger) - 반 고흐의 구두 해석

> "낡은 구두의 안쪽이 어둡게 열려 있는 데에서 노동하는 사람의 고단함이 내다보이고 있다. 뻣뻣한 구두의 무거움에는, 거친 바람이 부는 들판, 펀펀하게 뻗은 이랑을 오래도록 거닌 걸음의 끈질김이 있다. 가죽에는 흙의 축축함과 비옥함이 배어 있다. 구두창 밑으로는 땅거미 내릴 무렵의 들길의 외로움이 지난다. 구두에는 땅의 고요한 부름, 익어 가는 곡식의 조용한 선물과 겨울 들판의 적막함 속에 있는 알 수 없는 거부가 흔들리고 있다. 이 물건에는 양식의 확실성에 대한 불안과 궁핍을 또 한 번 이겨 낸 데 대한 말없는 기쁨과 아기의 태어남에 앞서서의 근심과 어디에나 있는 죽음의 위협에 대한 두려움이 흐르고 있다. 이 물건은 땅의 것이면서 농촌 아낙네의 세계 속에 간직되어 있다."

### 2.5 미셸 푸코 (Michel Foucault)

> "두 개의 이미지나 관념이 묶일 수 있는 구극적인 근거는 오로지 '사물의 쌍둥이 같은 성질'(**미셸 푸코**)에 있다."

## 3. 핵심 개념어

| 개념 | 설명 |
|------|------|
| **아름다움** | 사물의 내재적 성질이 아닌 사물 간의 관계에서 발생하는 사건 |
| **느낌** | 사람이 세계에 대하여 있는 한 방식 |
| **행복** | 세계에 거주하는 방식, 아름다움의 약속 |
| **땅과 세계** | 하이데거의 개념, 예술은 땅과 세계의 투쟁에서 발생 |
| **데포르마시옹(deformation)** | 개성적 삶을 통하여 매개되는 사물의 변형 |
| **공동 주체성** | 시대가 가진 이상적 삶, 세계의 주체적 극(極) |
| **전형적 인물** | 공동 주체성에 대한 탐구의 결과 |
| **통일성** | 예술 작품의 형식적 요건, 익숙한 세계의 원리 |

## 4. 글의 논지 구조

```
1. 서론: 아름다움의 본질
   └── 에머슨 「낱과 모두」 인용
   └── 아름다움 = 여러 가지 것이 어울려 있는 상태
   └── 사물 간의 관계에서 발생하는 사건

2. 느낌과 세계의 관계
   └── 느낌 = 사물에 '대한' 느낌
   └── 세계 = 서로 거울처럼 마주 비치는 사건들
   └── 비침의 관계 → 감정으로 드러남

3. 긴장감과 고양감
   └── 스탕달 『앙리 브륄라르의 생애』 인용
   └── 주관의 작용과 세계의 사건의 내적 합치
   └── "살아 있다는 것이 행복했다"

4. 행복과 세계 거주
   └── 행복을 통한 세계 거주
   └── 세계로부터의 소외 vs 세계의 둘러쌈
   └── 아름다움 = 행복의 약속

5. 지각의 구성 작용
   └── 플라톤 『티마이오스』: 눈에서 빠져나가는 빛
   └── 지각 = 수동적 기록이 아닌 적극적 구성
   └── 아름다움의 지각 = 정상적 지각의 고양

6. 하이데거의 예술론
   └── 반 고흐의 구두 해석
   └── 미적 대상 = 인간의 대자연, 대역사적 존재 방식
   └── 존재론적 조건으로서의 사람과 세계의 상호 삼투

7. 비유와 세계
   └── 미셸 푸코: "사물의 쌍둥이 같은 성질"
   └── 비유를 통한 세계의 전체적 모습 드러남
   └── 18세기 영시, 이조 시대 시조의 예

8. 문학의 세계
   └── '프로스트의 세계', '서정주의 시 세계'
   └── '디킨스의 세계', '제임스 조이스의 세계'
   └── 대작가 vs 소작가의 구분

9. 예술의 충동
   └── 하이데거: 땅과 세계의 투쟁
   └── 삶의 터를 살 만한 곳으로 확인하려는 충동
   └── 질서와 행복의 모티프

10. 통일성과 개성
    └── 통일성 = 익숙한 세계의 원리
    └── 개성적 삶을 통한 데포르마시옹
    └── 위대한 예술가 = 포용성 있는 개성

11. 공동 주체성
    └── 세계 = 개체적 삶을 넘어서는 공동의 장
    └── 시대의 테마에 대한 변주
    └── 전형적 인물 = 공동 주체성 탐구

12. 결론: 세계의 동적 이해
    └── 세계 = 생성 변화하는 과정
    └── 통일성 = 창조력의 지속성
    └── 익숙한 세계 = 창조적 전개의 유구함과 행복
```

## 5. 아름다움의 구조

| 요소 | 내용 |
|------|------|
| **에머슨의 통찰** | 아름다움은 사물의 내재적 성질이 아닌 관계에서 발생 |
| **느낌의 성격** | 제멋대로가 아닌, 세계에 대한 존재 방식 |
| **형이상학적 토대** | 세계 = 서로 거울처럼 마주 비치는 사건들 |
| **행복과의 관계** | 아름다움 = 행복의 약속 |

## 6. 하이데거 예술론 요약

| 개념 | 내용 |
|------|------|
| **반 고흐의 구두** | 노동, 들판, 흙, 외로움, 불안, 기쁨, 죽음의 연관 |
| **미적 대상** | 인간의 대자연(對自然), 대역사적(對歷史的) 존재 방식 |

## 7. 문학론적 명제

| 명제 | 내용 |
|------|------|
| 아름다움의 본질 | "아름다움은 어떤 사물의 내재적인 성질이 아니라 사물 간의 일정한 관계에서 발생하는 사건이다" |
| 느낌의 성격 | "느낌은 사람이 세계에 대하여 있는 한 방식이다" |
| 행복과 세계 | "행복을 통하여 우리는 세계에 거주한다" |
| 아름다움과 행복 | "적어도 아름다움은 행복의 약속으로 생각될 수 있다" |
| 예술의 충동 | "예술의 충동은 삶의 터를 근본적으로 자기가 살 만한 곳으로 확인하고자 하는 충동이라고 말할 수 있다" |
| 세계의 참뜻 | "세계의 참뜻은 그것이 개체적인 삶을 넘어서는 공동의 장이라는 데 있다" |
| 삶의 변주 | "우리의 개체적 삶은 시대가 가지고 있는 테마에 대한 변주에 불과하다" |
| 세계의 통일성 | "세계의 통일성은 총계가 아니라 전개 과정의 동일성, 창조력의 지속성이다" |
| 익숙한 세계 | "진정한 의미에서 그 익숙한 세계는 사람의 끊임없이 움직이는 삶의 전개와 동시에 전개되는 세계이다" |

## 8. 문학 세계의 예시

| 장르 | 예시 |
|------|------|
| **시** | '프로스트의 세계', '서정주의 시 세계' |
| **소설** | '디킨스의 세계', '제임스 조이스의 세계' |
| **시대적 시** | 18세기 영시의 전아(典雅)한 문화, 이조 시대 시조 |
| **대작가/소작가** | 작품이 하나의 세계를 이루느냐 못 이루느냐의 구분 |

## 9. 개성과 세계의 관계

| 차원 | 내용 |
|------|------|
| **개인적 개성** | 삶의 스타일, 데포르마시옹의 특이한 방법 |
| **괴팍한 개성** | 작은 의미의 예술가 |
| **포용적 개성** | 위대한 예술가의 특징 |
| **공동 주체성** | 시대의 이상적 삶, 전형적 인물 |

## 10. 전형적 인물의 특징

| 특징 | 내용 |
|------|------|
| **정의** | 공동 주체성에 대한 탐구 결과 |
| **성격** | 대개 혁명적 인물, 비극적 인물 |
| **이유** | 어떠한 현실도 가능성을 그대로 허용하지 않음 |
| **동적 이해** | 삶의 가능성으로서 주어진 총화를 넘어감 |
