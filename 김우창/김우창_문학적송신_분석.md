## 3. 인용 예시문

### 3.1 케네스 버크 (Kenneth Burke)

> "**케네스 버크(Kenneth Burke)**가 시를 상징 행동이라고 말한 것도 이러한 뜻으로 해석될 수 있다. 문학적 전달에 있어서 우리는 객관적 정보를 전달해 받는 것이 아니라, 그러한 정보를 하나의 주체적인 행동자의 입장에서 재연(再演)해 보는 것이다."

### 3.2 기욤 아폴리네르 (Guillaume Apollinaire)

> "**기욤 아폴리네르(Guillaume Apollinaire)**가
>
> 내 이성(理性)의 들녘 푸른 가파름에
> 홀로 비껴 선 석양의 너도밤나무……
>
> 라고 할 때 그것은 시에 솟아 있는 나무가 외부 공간에 있으면서도 또 내면에 있으며, 우리들의 내면을 연결해 주는 이성의 장(場)에 있음을 말한 것일 것이다."

## 4. 분석 작품: 신동집 「송신」

### 4.1 전문 인용

> 바람은 한로(寒露)의
> 음절(音節)을 밟고 지나간다.
> 귀뚜리는 나를 보아도
> 이젠 두려워하지 않는다.
> 차운 돌에 수염을 착 붙이고
> 멀리 무슨 신호(信號)를 보내고 있다.
> 어디선가 받아 읽는 가을의 사람은
> 일손을 놓고
> 한동안을 멍하니 잠기고 있다.
> 귀뚜리의 송신(送信)도 이내 끝나면
> 하늘을 알 수 없는
> 청자(靑瓷)의 심연(深淵)이다.

### 4.2 김우창의 해석

| 요소 | 해석 |
|------|------|
| **귀뚜리의 신호** | 시인 자신의 송신에 비교 |
| **내용 없는 송신** | 비어 있으면서 의미에로의 형성을 암시 |
| **가을의 공간** | 동양 시인들의 여백(餘白)의 공간 |
| **청자의 심연** | 건너뛸 수 없는 공간, 하늘의 심연 |
| **뛰어난 점** | 여백의 공간을 침묵으로 전달, 감상으로 메꾸지 않음 |

## 5. 핵심 개념어

| 개념 | 설명 |
|------|------|
| **송신(送信)** | 시인의 시적 전달 행위, 내용 없는 신호 |
| **침묵의 공간** | 모든 시의 본질적 전달, 비어 있는 공간의 제시 |
| **여백(餘白)** | 동양 시인들이 전달해 온 공간, 가을 속의 느낌 |
| **의미 발생의 과정** | 시인이 전달하고자 하는 것, 마음의 경이(驚異) |
| **생략법(省略法)** | 연계 관계를 빼어 버리는 시인의 기술 |
| **마음의 다층적 구조** | 의식의 여러 평면, 체계와 체계 간의 상호 작용 |
| **상징 행동(symbolic action)** | 케네스 버크의 시 정의 |
| **초월적 자아** | 개인적 의식을 넘어서는 사회적 초월 자아 |
| **이성(理性)의 장(場)** | 우리들의 내면을 연결해 주는 공간 |

## 6. 글의 논지 구조

```
1. 서론: 신동집의 「송신」
   └── 귀뚜리의 신호 = 시인의 송신
   └── 내용 없는 송신
   └── 가을의 공간, 신비감
   └── 의미 가능성에로의 형성

2. 「송신」의 분석
   └── "바람은 한로의 음절을 밟고"
   └── 귀뚜리와 시인의 공존(共存)
   └── 수신자의 참여
   └── 청자(靑瓷)의 심연

3. 침묵의 공간
   └── 여백의 공간 전달
   └── 감상으로 메꾸지 않는 뛰어남
   └── 모든 시의 본질적 전달 = 침묵
   └── 의사전달 밑에 깔린 공간

4. "코끼리는 전염성이 강하다" 분석
   └── 불합리한 진술의 억압
   └── 마음의 용해 작용
   └── 해석의 평면 조정
   └── 유추를 통한 확대 적용

5. 생략법과 의미 발생
   └── 연계 관계의 생략
   └── 의식의 넓은 공간
   └── 마음 = 의미 발생의 창조적 근원
   └── 마음의 경이(驚異)

6. 마음의 다층적 구조
   └── 사고 체계 간의 상호 작용
   └── 사泡의 체계 vs 유희의 체계
   └── 한 층에서 다른 층으로의 초월
   └── 의미의 구역 설정과 개척

7. 존재의 공간과 마음의 공간
   └── 마음의 활동 범위 = 존재의 공간
   └── 공존(共存)의 장(場)으로서의 존재 공간
   └── 시적 진술의 자기 지시성

8. 문학적 전달의 본질
   └── 객관적 정보가 아닌 마음의 과정
   └── 춤의 비유: 내용 없는 전달
   └── 세계라는 큰 바탕에 근거
   └── 공동의 세계 구축

9. 소설과 연극의 전달
   └── 주인공의 관점 → 독자의 관점
   └── 공동의 세계 확인
   └── 마음 = 활동
   └── 케네스 버크: 시 = 상징 행동

10. 초월적 자아와 이성의 장
    └── 주체적 마음의 일치
    └── 초개인적 의식에 의한 소유
    └── 사회적 초월 자아
    └── 아폴리네르: 이성의 들녘
```

## 7. 시적 전달의 단계

| 단계 | 내용 |
|------|------|
| **1단계: 침묵의 공간** | 비어 있는 공간의 제시, 여백의 전달 |
| **2단계: 생략법** | 연계 관계의 생략, 의식의 공간 확대 |
| **3단계: 의미 발생** | 마음의 창조적 근원에서 의미 생성 |
| **4단계: 체계 간 상호작용** | 다층적 평면 간의 이동과 초월 |
| **5단계: 공동 세계 구축** | 주관과 주관의 일치, 초월적 자아 |

## 8. 문학론적 명제

| 명제 | 내용 |
|------|------|
| 시의 본질적 전달 | "어떤 의미에 있어서 모든 시의 본질적인 전달은 침묵의 전달이며, 비어 있는 공간의 제시이다" |
| 생략법의 취지 | "시인의 생략법이 우리에게 일깨워 주는 것은 이러한 원초적(原初的)인 사실일 것이다. 즉 우리는 코끼리와 전염성 사이를 잇는 사이에 의식의 넓은 공간을 접하게 된다" |
| 시인의 전달 목적 | "시인이 전달하고자 하는 것은 단순한 의미 내용이 아니라 의미 발생의 과정 자체이며, 그 과정을 통하여 마음의 경이(驚異)를 표현하고자 하는 것이다" |
| 문학의 전달 | "문학에 있어서 보다 근원적인 전달은 객관적인 정보 자체보다 그러한 정보를 구성하는 마음의 과정이다" |
| 상징 행동 | "문학적 전달에 있어서 우리는 객관적 정보를 전달해 받는 것이 아니라, 그러한 정보를 하나의 주체적인 행동자의 입장에서 재연(再演)해 보는 것이다" |
| 마음의 일치 | "문학적 전달은—사실상 사람과 사람 사이의 모든 전달은 하나의 주체적인 마음과 또 하나의 주체적인 마음의 일치에서 성립한다" |
| 시인의 일 | "시인이 하는 일 가운데 하나는 이러한 공동의 공간을 확인하고 또 확대해 가는 일이다. 이러한 공간은 부단히 은폐되고 또 좁아지기 때문이다" |
| 시의 이성 | "시가 전달하는 것은 우리들의 생활 한복판에 있는 바, 더러는 찌그러지고 더러는 활달한 우리들의 마음, 우리 사회의 마음인 것이다. 그것은 다시 말하면 한 사회가 지닌 가장 넓은 의미에서의 이성(理性)과 같다" |

## 9. "코끼리는 전염성이 강하다" 분석

| 분석 단계 | 내용 |
|-----------|------|
| **1. 첫인상** | 불합리, 마음의 억압 |
| **2. 해석 시도** | 전염성 개념의 확대 적용 |
| **3. 유추** | 세균 → 고등동물 → 인간(인구 폭발) |
| **4. 체계 간 상호작용** | 사泡의 체계(위험) vs 유희의 체계(동물원) |
| **5. 의미** | 의미 발생의 과정 자체, 마음의 경이 |
| **6. 결론** | "코끼리는 전염성이 강하다"는 그 말 이외의 아무것도 의미하지 않음 |
