# 김우창 「말과 현실 - 국어 순화운동에 대한 몇 가지 생각」 분석

## 글 정보
- **저자**: 김우창
- **발표**: 1976년
- **성격**: 언어론, 사회비평

## 1. 인용 철학자/언어학자/작가

| 인물 | 인용 맥락 |
|------|-----------|
| **메를로퐁티(Merleau-Ponty)** | 언어에 대한 독창적 통찰, 소쉬르의 기호론 수용 |
| **소쉬르(Ferdinand de Saussure)** | 기호는 다른 기호와의 연관 관계 속에서만 의미를 가짐 |
| **자크 라캉(Jacques Lacan)** | 정신 분석 = 자유롭고 창조적인 언어 기능의 회복 |
| **폴 클로델(Paul Claudel)** | "우유의 흰빛은 검은 색을 안에 받쳐 가지고 있다" - 시적 직관 |
| **뷔퐁(Georges-Louis Buffon)** | "스타일은 사람이다" |

## 2. 인용 예시문

### 2.1 소쉬르 (Ferdinand de Saussure)

> "그의 언어 철학에서 가장 중요한 명제는 어떠한 기호는 반드시 다른 기호와의 연관 관계 속에서만 의미를 갖는다는 **소쉬르(Ferdinand de Saussure)**의 생각이었다. 기호는 그 자체로는 아무 의미를 갖지 않고 다만 그 자체와 다른 기호 사이에 존재하는 의미상의 편차만을 표시한다."

### 2.2 자크 라캉 (Jacques Lacan)

> "근래에 프로이트가 창안한 정신 분석의 목적은 단순히 정신의 병을 고치려는 것이 아니라 환자로 하여금 자유롭고 창조적인 언어 기능을 회복하게 하려는 것이라는 학자도 있거니와(가령 프랑스 **자크 라캉(Jacques Lacan)** 박사와 같은 사람의 새로운 통찰의 하나는 이런 것으로 생각된다.)"

### 2.3 폴 클로델 (Paul Claudel)

> "보다 인상적인 것은 **폴 클로델(Paul Claudel)**의 '우유의 흰빛은 검은 색을 안에 받쳐 가지고 있다.'라는 시적 직관이다."

### 2.4 뷔퐁 (Georges-Louis Buffon)

> "'스타일은 사람이다.'라는 **뷔퐁(Georges-Louis Buffon)**의 진부한 말은 역시 진리이다. 한 나라의 말은 곧 사회가 추구하는 인간 이념에 대응한다고 말할 수 있다."

## 3. 핵심 개념어

| 개념 | 설명 |
|------|------|
| **국어 순화 운동** | 외국어·외래어 추방을 중심으로 한 민족주의적 언어 운동 |
| **문화적 위화감** | 문화적 표현의 부조화에 대한 민감함 |
| **의미의 울림** | 낱말 뒤에 깃든 말 전체에 퍼지는 소리와 의미 |
| **창조적 언어 능력** | 기호와 사물을 종합·분석하여 의미를 생성하는 능력 |
| **언어 공동체** | 개인과 역사적 기억을 공유하는 공동 능력의 주체 |
| **아어화(雅語化)** | 한자어 대신 고운 우리말을 쓰려는 인위적 경향 |
| **감정의 상투화** | 자동적이고 정해진 감정 반응의 강요 |
| **스타일** | 사람됨의 표현, 사회가 추구하는 인간 이념 |

## 4. 글의 논지 구조

```
1. 서론: 국어 순화 운동의 당위성
   └── 민족주의적 경향의 표현
   └── 관권 개입에 대한 우려
   └── 맹목적 민족주의 경계

2. 문화적 위화감의 본질
   └── 양복에 갓과 고무신의 예
   └── 관습 vs 삶에 대한 본능적 이해
   └── 세계를 조화된 일체로 파악하려는 욕구

3. 언어의 구조와 의미
   └── 낱말의 의미 = 언어 구조 속 관계
   └── 의미의 울림 = 수평적·역사적 연관
   └── 외래어 사용 → 언어 세계의 빈곤화

4. 소쉬르와 메를로퐁티의 언어론
   └── 기호 = 다른 기호와의 편차
   └── 의미 = 언어 전체에 내재
   └── 언어 습득 = 대칭·반대·평형 관계 식별 능력

5. 창조적 언어 능력
   └── 개인적 능력 + 언어 공동체의 공동 능력
   └── 역사적 기억의 일깨움
   └── 동정적 참여를 통한 창조

6. 언어와 세계의 소유
   └── 언어를 통한 세계·사회 구성
   └── 의미 창조 = 세계 소유
   └── 언어의 창조적 자유 = 인간의 자유

7. 국어 순화의 확대된 의미
   └── 외래어 추방만으로 완성되지 않음
   └── 언어의 창조적 사용
   └── 언어와 세계의 변증법적 교호 관계

8. 치유의 기술
   └── 시 = 치유의 기술
   └── 정신 분석 = 창조적 언어 기능 회복 (라캉)
   └── 정치의 기술

9. 언어와 사회적 지위
   └── 흑인 호칭의 변화: 니그로 → 블랙
   └── 상인 호칭의 변화: 장사치 → 경제인
   └── 식모 → 가정부
   └── 세종대왕의 한글 반포 취지

10. 아어화의 문제점
    └── 경험과 표현의 범위 축소
    └── 안일주의와의 결탁
    └── '몸담고 있다' = 이중의 소외

11. 언어의 타락 현상
    └── '밝히다'의 오용 = 도그마의 시대
    └── '얼', '슬기' = 신비화 과정
    └── 상냥한 광고문 = 반진반위(半眞半僞)

12. 감정의 상투화
    └── 신문 기사의 감정 묘사
    └── 감정의 자동인형화
    └── 강요된 윤리 vs 바른 윤리

13. 결론: 언어의 참다운 기능
    └── 뷔퐁: "스타일은 사람이다"
    └── 언어의 혼란 = 인간상의 불분명
    └── 진실의 전달, 자유로운 의사 교환
    └── 우리말 사용 역사 100년 미만
```

## 5. 언어론적 명제

| 명제 | 내용 |
|------|------|
| 언어의 구조 | "어떠한 낱말도 그 자체로서 하나의 고정된 의미를 갖고 있지는 않다. 낱말의 모양이나 뜻은 그것이 속해 있는 언어의 구조 속에서 다른 말들과 대조 또는 유사(類似) 관계를 통하여 결정된다" |
| 의미의 울림 | "한마디의 낱말 뒤에는 말 전체에 퍼지는 소리와 의미의 울림이 깃들어 있다" |
| 의미 창조 | "사물과 언어의 의미는 외부로부터 주어지는 것이 아니라 우리 스스로에 의하여 창조되는 것이다" |
| 소쉬르의 명제 | "기호는 그 자체로는 아무 의미를 갖지 않고 다만 그 자체와 다른 기호 사이에 존재하는 의미상의 편차만을 표시한다" |
| 언어와 세계 | "세계가 열리고 사회가 이루어지는 것은 언어를 통하여서이다" |
| 언어와 자유 | "언어의 창조적 자유를 얻는 것은 인간 자신이 자유롭게 되는 것과 병행한다" |
| 스타일의 의미 | "'스타일은 사람이다.'라는 뷔퐁의 진부한 말은 역시 진리이다. 한 나라의 말은 곧 사회가 추구하는 인간 이념에 대응한다" |

## 6. 국어 순화 운동 비판

### 6.1 긍정적 측면

| 측면 | 내용 |
|------|------|
| **당위성** | 외세 지배 과거에 대한 필연적·필요한 반작용 |
| **민족주의** | 정치적·경제적 외부 세력으로부터의 자립 의지 |
| **창조적 근원** | 우리말이 창조적 언어 능력의 근원이 되어야 함 |

### 6.2 비판적 측면

| 측면 | 내용 |
|------|------|
| **맹목적 민족주의** | 토론과 생각을 중단케 하는 것 |
| **지엽적 문제** | 코너킥/구석차기는 근본적 질의 차이 없음 |
| **아어화 경향** | 경험과 표현의 범위 축소, 안일주의 |
| **감정의 상투화** | 자동적 감정 반응 강요 |

## 7. 언어의 타락 사례

| 사례 | 분석 |
|------|------|
| **'몸담고 있다'** | 직업에 완전히 싸여 자기 객체화에 자랑을 느끼는 표현 |
| **'밝히다'의 오용** | 강요된 도그마의 시대를 상기 |
| **'얼', '슬기'** | 막연한 외경감·신비감으로 정확한 이해와 사고를 막음 |
| **상냥한 광고문** | 반진반위(半眞半僞)의 멍멍한 상태 조장 |
| **감정 묘사** | "울먹였다", "활짝 웃었다" = 감정의 자동인형 |

## 8. 사회적 지위와 언어 변화

| 대상 | 호칭 변화 | 의미 |
|------|-----------|------|
| **흑인** | 니그로 → 컬러드 피플 → 블랙 | 인간적 지위 획득 |
| **상인** | 장사치 → 장사꾼 → 상인 → 사업가 → 경제인 | 사회적 지위 상승 |
| **가사 노동자** | 식모 → 가정부 | 봉건적 관계 → 임금 노동자 |
