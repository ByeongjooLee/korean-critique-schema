# 김우창 「작은 것들의 세계 - 피천득론」 분석

## 글 정보
- **저자**: 김우창
- **발표**: 1976년
- **분석 대상**: 피천득(琴兒, 금아 선생)의 수필

## 2. 인용 문학가/사상가

| 인물 | 인용 맥락 |
|------|-----------|
| **탈레스(Thales)** | 별을 쳐다보다 도랑에 빠진 철학자 - 먼 것만 보는 것의 위험 |

---

## 3. 인용 예시문

### 3.1 탈레스 (Thales)

> "그러나 별을 쳐다보고 땅 위를 걸어가던 **탈레스**가 도랑에 빠졌다는 것은 유명한 이야기이지만, 허허한 공간의 펼쳐짐과 몇백만 년의 지질학적 시간만을 생각하면서 산다든지 사회와 역사의 차원만을 유일한 현실로 받아들이면서 산다는 데에는 무엇인가 허황스럽고 비인간적인 것이 있다고 생각될 수도 있다."

### 3.2 도연명 (陶淵明)

> "**도연명(陶淵明)**처럼 삶의 근본으로 돌아가는 일도 큰 이상을 위하여 자기를 버리는 일 다음으로 어려운 것이다."

### 3.3 워즈워스 (Wordsworth)

> "금아 선생의 아름다움이 어린 시절에 뿌리내리고 있다는 점에서 그것이 약간은 퇴행적인 것이라 할 사람이 있을는지 모른다. 그러나 선생에게 어린 시절은 퇴행의 피난처라기보다는 **워즈워스에게 그러했던 것처럼** 하나의 이상이 된다."

### 3.4 솔제니친/이반 데니소비치

> "한 그릇의 따뜻한 국과 따뜻한 털장화에 대한 관심이 없었다면, **이반 데니소비치**는 강제 수용소의 혹독함을 견디어 내지 못하였을 것이다."

## 4. 김우창의 평가 분석 (긍정/부정/중립)

### 4.1 긍정적 평가

| 평가 내용 | 근거 |
|-----------|------|
| 독자적인 세계 | "결코 협소한 세계는 아니면서, 선생만의 독자적인 세계를 이룬다" |
| 건강한 세계 | "그 나름으로의 건강은 이 세계의 특징이다" |
| 비극적 인식 | "신록의 싱싱한 생명이 죽음으로 하여 더욱 찬란해지는 것을 아는 선생의 비극적 인식" |
| 강한 긍정의 의지 | "너무나 험한 시대를 살아감에 절실히 요구되는, 강한 긍정에의 의지의 표현" |
| 원초적 진실에 충실 | "삶의 원초적 진실에 충실하고 현대의 모든 가상(假象)을 꿰뚫어 보는 어쩌면 영웅적일 수도 있는 노력" |
| 사랑의 공간 | "이 세계의 중요성은 무엇보다도 그것이 사랑의 공간이라는 데 있다" |
| 미문(美文) | "선생의 글은 과연 산호나 진주와 같은 미문(美文)이다" |
| 현대 수필의 번설성 극복 | "금아 선생의 수필이 현대 수필의 번설성(煩剧生)을 벗어나 삶의 한 국면을 밝혀 주고 있는 것은 참으로 희귀한 일" |
| 목가적 이상 | "우리 마음 깊이 자리 잡고 있는 목가적(牧歌的) 이상을 상기시켜 준다" |

### 4.2 부정적 평가 (한계 지적)

| 평가 내용 | 근거 |
|-----------|------|
| 소박함의 한계 | "더러는 그것이 너무 소박한 것인 때도 있으나" |
| 착잡한 전모의 미제시 | "금아 선생의 글이 우리 삶의 착잡한 모습의 전모를 들추어내 주는 것은 아니라 할는지 모르나" |

### 4.3 중립적 평가 (분석적 서술)

| 평가 내용 | 근거 |
|-----------|------|
| 작은 것의 세계 | "금아 선생의 수필의 세계는 나날의 세계... 아름다운 작은 일들로 이루어진다" |
| 덧없음의 인식 | "그 아름다움이 덧없는 것이며 끊임없는 소멸의 위험에 있음을 아쉬워하는 마음" |
| 제약 속의 보람 | "평범한 사람에게 주어진 대로의 삶이 근본적으로 제약 속에 있는 삶이며 그럼에도 불구하고 이 제약 속에서일망정 평범한 삶도 그 나름으로 보람 있는 삶이어야 한다" |
| 무상적 증여 | "사물이나 사람에 대한 우리의 관계가 원칙적으로 무상적(無償的) 증여(贈與)의 것이어야 마땅하다" |

### 4.4 종합 평가

김우창의 피천득 평가는 **작은 것의 의미를 철학적으로 정당화하면서 그 가치를 옹호하는 비평**이다:

1. **원근법적 서론**: 우주적 시간과 공간에서 나날의 삶까지의 원근법을 통해 작은 것의 의미를 정당화한다.

2. **시대적 맥락화**: 험한 시대에 작은 것에 대한 관심이 "미치지 않고 살게 하는 유일한 지주"임을 강조한다.

3. **문학적 가치 인정**: 피천득의 수필이 "산호나 진주와 같은 미문"이며 수필의 본래적 정신에 부합함을 평가한다.

4. **사랑의 철학 발견**: 어머니-딸에 대한 사랑에서 "사랑의 공간화"와 "지킴의 거리"라는 독특한 인간관계의 이상을 발견한다.

5. **목가적 이상의 옹호**: "안에다 가꾸는 꿈의 공간"의 중요성을 강조하며 결론짓는다.

## 5. 핵심 개념어

| 개념 | 설명 |
|------|------|
| **작은 것들의 세계** | 나날의 삶에서 겪는 아름다운 작은 일들 |
| **원근법** | 먼 것(우주, 역사)과 가까운 것(나날의 삶)의 균형 |
| **운명에 대한 사랑** | 제약 속에서 삶의 보람을 찾는 태도 |
| **무상적 증여(無償的 贈與)** | 대가 없이 주는 관계 |
| **사랑의 공간화** | 멀리서 지켜보며 자유와 신뢰의 공간을 마련함 |
| **지킴의 거리** | 억누르지 않는 자유롭고 허용하는 관계 |
| **목가적 이상** | 온화한 행복의 모습 |

## 6. 글의 논지 구조

```
1. 서론: 원근법의 문제
   └── 우주적 시간/공간에서 본 인간
   └── 역사와 사회의 테두리
   └── 나날의 삶의 중요성
   └── 작은 것에의 사랑: 삶의 지주

2. 금아 선생의 세계
   └── 나날의 세계, 아름다운 작은 일들
   └── 깨끗하고 부드럽고 조촐한 느낌
   └── 젊음과 봄에 대한 찬미

3. 아름다움의 성격
   └── 도려냄이 아닌 긍정의 동경
   └── 덧없음과 애수
   └── 비극적 인식

4. 제약과 운명
   └── 평범한 삶의 제약
   └── 운명에 대한 사랑
   └── 작은 것에서 보람 찾기

5. 시대와 긍정의 의지
   └── 시대의 험악함에서도 아름다움 지키기
   └── 강한 긍정에의 의지
   └── 원초적 진실에 충실함

6. 사랑의 공간
   └── 어린 시절: 워즈워스적 이상
   └── 어머니의 사랑: 허용의 공간
   └── 딸에 대한 사랑: 지킴의 거리

7. 무상적 증여
   └── 선물의 본질
   └── 소유와 수탈 없는 관계
   └── 인간관계의 자유로운 주고받음

8. 수필의 본질
   └── 평범한 사람의 평범함 존중
   └── 대화의 장
   └── 현대 수필의 번설성 극복

9. 결론: 목가적 이상
   └── 온화한 행복
   └── 사랑과 고마움의 공간
   └── 안에다 가꾸는 꿈의 공간
```

## 7. 피천득 수필의 특징 정리

| 특징 | 내용 |
|------|------|
| **소재** | 감각적 기쁨, 일상의 파문, 넓은 세계의 반사적 광영 |
| **미감** | 깨끗하고 부드럽고 조촐함, 건강함 |
| **선호** | 피아노 > 바이올린, 야생 백합 > 병든 장미, 시골 처녀 > 모나리자, 새벽 > 황혼 |
| **정서** | 긍정에의 동경, 덧없음의 애수, 비극적 인식 |
| **인간관계** | 무상적 증여, 억눌림 없는 자유, 지킴의 거리 |
| **문체** | 평이하고 일상적, 곱고 간결한 우리말, 미문(美文) |

## 8. 사랑의 공간화 분석

| 관계 | 내용 |
|------|------|
| **어머니의 사랑** | 숨바꼭질에서 짐짓 못 찾는 척, 딴 구슬 돌려주기, 남모르게 눈물 - 유희와 욕망과 성장의 공간 마련 |
| **딸에 대한 사랑** | 멀리서 일상적 동작 지켜보기, 학교 유리창 너머 바라보기 - 신뢰와 자유를 익힐 공간 |
| **특징** | 감정의 근접이 아닌 거리, 억누르지 않는 허용, 사랑의 공간화 |

## 9. 수필 장르론

|------|------|

| **현대의 위기** | 담소로 문제 해결 불가능, 신변잡사 요설로 전락 위험 |

## 10. 원근법의 구조

```
우주적 시간/공간 (가장 먼 것)
    ↓
지질학적 시간
    ↓
세계/문명의 역사
    ↓
민족의 삶/역사와 사회
    ↓
나날의 삶 (가장 가까운 것)
    ↓
작은 것들의 세계 (피천득의 영역)

→ 문학의 기능: "우리의 눈을 적당한 인간적인 높이에 유지"
```

## 11. 인용 중국 시인

| 인용 | 출처 |
|------|------|
| "득료애정통고(得了愛情痛苦) / 실료애정통고(失了愛情痛苦)" | 젊어서 죽은 중국 시인 (피천득이 모래 위에 쓴 글귀) |

- **의미**: 사랑을 얻어도 고통, 사랑을 잃어도 고통
- **맥락**: 신록의 싱싱한 생명이 죽음으로 더욱 찬란해지는 비극적 인식
