# 김우창 「주체의 형식으로서의 문학 - 작품해석의 전제에 대한 한 성찰」 분석

## 글 정보
- **저자**: 김우창
- **발표**: 1975년
- **성격**: 문학 해석학 이론, 독서론

## 1. 분석 대상

| 주제 | 내용 | 비고 |
|------|------|------|
| **독서 현상** | 문학 해석의 출발점으로서의 독서 | 발생적 관점 |
| **문학적 전달** | 체험의 전달 가능성 문제 | 해석학적 전제 |
| **주체적 이해** | 인간의 실천적 주체성과 문학 해석 | 존재론적 토대 |

## 2. 인용 철학자/사상가

| 인물 | 인용 맥락 |
|------|-----------|
| **딜타이(Dilthey)** | '이해(理解)', '감정 이입(感情移入)' - 주체적 관점의 교환을 통한 문학적 전달 |
| **실존주의자들** | 투기성(投企性) - 스스로를 던져 의지하고 계획하고 실천하는 인간 본질 |
| **카렐 코지크(Karel Kosik)** | '구체적 전체성' - 부분과 전체의 상호 작용적 창조 |

---

## 3. 인용 예시문

### 3.1 딜타이 (Dilthey)

> "문학적 전달이 이러한 관점의 동정적(同情的)인 교환을 필요로 한다는 것은 **딜타이** 이후의 미학자들이 이미 '이해(理解)'라든가 '감정 이입(感情移入)'이라는 말로 설명한 바 있다."

### 3.2 실존주의자들

> "보다 적극적으로 **실존주의자들**이 말하듯이 스스로를 던져 의지하고 계획하고 실천하는 투기성(投企性)이 인간의 본질을 이룬다는 의미에서이다."

### 3.3 카렐 코지크 (Karel Kosik)

> "**카렐 코지크(Karel Kosik)**에 의하면, 이 전체성 안에서 '부분은 서로서로 또 전체와 안으로 연결되고 맺어져 있다. 부분은 추상적으로 부분 위에 덮어씌워질 수 없다. 전체는 부분의 상호 작용에서 창조된다.' 또 '전체는 이미 만들어져 있는 모두로서 거기에 부분의 속성이나 관계와 내용이 채워져 있는 것이 아니다. 전체 그것이 구체화 속에 나타난다. 구체화는 다만 내용의 창조가 아니고 전체의 창조도 뜻한다.'"

## 4. 핵심 개념어

| 개념 | 설명 |
|------|------|
| **사후 추적 행위(事後追跡行爲)** | 의식 활동이 이미 앞서 가는 삶을 새로운 통일성 속에 거둬들이려는 것 |
| **체험** | 인간적 관련 속에서 파악된 사실의 수용 |
| **주체적 이해** | 사물을 자신의 입장에서 재구성하는 주체의 입장에서 파악하는 것 |
| **이해(理解)/감정 이입(感情移入)** | 딜타이 이후 미학의 핵심 개념, 관점의 동정적 교환 |
| **투기성(投企性)** | 스스로를 던져 의지하고 계획하고 실천하는 인간의 본질 |
| **초월적 주체** | 개개의 주체를 포함하는 보다 큰 생명 충동 또는 역사적 실현 |
| **구체적 전체성** | 부분과 전체가 상호 작용을 통해 창조되는 전체성 (코지크) |
| **주제적 발전** | 테마의 반복을 통한 새로운 의미와 전체적 구조의 형성 |
| **스타일** | 사물 자체의 지속 원리이자 인간이 사물에 선택을 행한 역사적 자취 |

## 5. 글의 논지 구조

```
1. 서론: 의식과 삶의 관계
   └── 의식 = 사후 추적 행위
   └── 삶의 자기 회복 운동
   └── 비평 활동에서의 상기 필요성
   └── 독서 현상에서의 출발

2. 독서와 전달의 문제
   └── 전달의 문제로서의 독서
   └── 과학적 진술 vs 문학적 진술
   └── 체험을 시금석으로 한 검증
   └── 두 가지 체험의 관계

3. 체험과 전체성
   └── 문학 작품의 전체성
   └── 단편적 사상(事象)의 종합화
   └── 인간적 관점에서의 종합 = 체험
   └── 개인에 특유한 종합의 전달 어려움

4. 주체적 이해의 조건
   └── 인간적 관점의 동일성
   └── 딜타이: 이해, 감정 이입
   └── 인간의 동질성과 사회성
   └── 사회적·역사적 주체성의 매개

5. 이해의 유추성과 역사성
   └── 상사적(相似的) 일치
   └── 완전한 이해의 불가능성
   └── 실존주의: 투기성(投企性)
   └── 개체의 역사적 궤적

6. 동질성과 차이의 변증법
   └── 유추적 간격의 역설
   └── 창조성과 편차
   └── 유기체 간의 상사 관계
   └── 전체적 동일성과 개체적 차이의 상보 관계

7. 초월적 주체와 체험의 가능성
   └── 초월적 주체의 매개
   └── 체험의 개연적 판단
   └── 가능성의 탐구로서의 문학
   └── 초월적 가능성의 한계 확인

8. 문학 해석의 전제
   └── 주체적 이해의 성립
   └── 삶 자체의 주체적 성격
   └── 문학 = 삶에 대한 해석
   └── 비평 = 구조의 구조 해명

9. 인간의 실천적 주체성
   └── 자유로운 자기실현의 이상
   └── 세계의 저항과 선택·결단
   └── 문제로서의 세계 파악
   └── 인간 자유의 균열에의 서식

10. 주제적 발전과 스타일
    └── 음악에서의 테마 반복
    └── 소설과 생애에서의 주제적 반복
    └── 스타일 = 지속의 원리 + 선택의 역사

11. 역사와 집단적 선택
    └── 다른 사람과의 관계
    └── 자유와 구속의 상호 작용
    └── 역사적 선택의 연쇄 구조
    └── 개체적 의지와 집단 의지의 융화

12. 결론: 구체적 전체성
    └── 카렐 코지크의 구체적 전체성
    └── 부분과 전체의 상호 작용적 창조
    └── 문학 = 구체적 전체성의 체험적 이해
    └── 문학 해석 = 구체적 전체성에의 모험
```

## 6. 주요 논제 분석

### 6.1 독서와 해석의 발생론적 관점

| 논제 | 내용 |
|------|------|
| **의식의 사후성** | 생각보다 앞서 존재하고 사는 것이 인간, 의식은 삶의 사후 추적 |
| **독서에서의 출발** | 문학 해석의 기도는 독서에서 출발, 독서 속 일어난 것을 밝히는 작업 |
| **전달의 복잡성** | 개체적 의식이 예술 매체를 통해 사회적·역사적 연속체로 매개되는 문제 |

### 6.2 체험과 전체성의 문제

| 논제 | 내용 |
|------|------|
| **체험의 정의** | 인간적 관련 속에서 파악된 사실의 수용 |
| **전체성의 부여** | 단편적 사상(事象)에 전체성을 부여하는 것이 체험의 원리 |
| **전달의 어려움** | 문학 작품의 전체성은 개개의 작품에 특유하여 전달이 어려움 |

### 6.3 주체적 이해의 조건

| 조건 | 내용 |
|------|------|
| **인간의 동질성** | 시작하고 만들고 변형하는 존재로서의 동질성 |
| **본래적 사회성** | 인간 상호 간의 개방성 |
| **역사적 매개** | 사회적·역사적 주체성의 동일함에 의한 매개 |

## 8. 문학론적 명제

| 명제 | 내용 |
|------|------|
| 의식과 삶 | "의식 작용이 사후 추적의 성질을 갖는다고 볼 때 우리가 시작할 수 있는 곳의 하나는 독서 현상 그것이다" |
| 체험의 시금석 | "문학적 진술에 있어서의 핵심적 사실은 체험이고 그 진실성 여부의 검증도 체험을 시금석(試金石)으로 하여 행해진다" |
| 주체적 이해 | "관점의 일치로부터 어떤 체험을 이해한다는 것은 인간에게 드러나는 사물을 객체적(客體的)으로 파악한다는 것이 아니라 그 사물을 자신의 입장에서 재구성하는 주체의 입장에서 파악한다는 것이다" |
| 문학의 기능 | "서사 문학의 주요한 기능은 단순히 개인적인 체험의 전달이라기보다는 체험의 가능성에 대한 탐구이고, 나아가 나와 내 이웃의 구체적인 체험과 다른 역사적인 체험을 통하여 투사(投射)되는 초월적 가능성의 한계를 확인하는 일이다" |
| 문학과 전체성 | "문학 작품은 이러한 구체적 전체성을 체험으로써 이해케 하려 한다. 그리하여 비록 '순진한 의지'의 주체적인 실현이 아닐망정 인간이 주체적으로 있으며 스스로의 세계의 창조에 참여하고 있다는 것을 우리에게 깨우쳐 주려고 한다" |
| 문학의 기적과 저주 | "문학 작품의 기적이며 저주는 이러한 전체성의 결여를 가지고도 하나의 전체성을 구축할 수 있다는 데 있다" |
| 문학 해석의 모험 | "문학의 해석도 그 반성을 통해서 작가와 마찬가지로 구체적 전체성에의 모험에 참여한다. 그리하여 인간의 참된 자유가 실천적으로 어떻게 구현될 수 있는가를 탐구한다" |

## 9. 실천적 주체성의 단계

| 단계 | 내용 |
|------|------|
| **1단계: 자유로운 자기실현** | 순진한 의지의 세계, 의지의 지속성과 행동의 통일성 |
| **2단계: 세계의 저항** | 질료가 아닌 반대 의지로서의 세계, 선택과 결단의 형식 |
| **3단계: 문제로서의 세계** | 세계 자체의 자기 분열, 인간 자유의 균열에의 서식 |
| **4단계: 주제적 발전** | 테마의 반복, 스타일의 형성, 사물과의 교섭 과정 |
| **5단계: 역사적 실천** | 집단적 선택, 개체적 의지와 집단 의지의 융화 |
| **6단계: 구체적 전체성** | 부분과 전체의 상호 작용적 창조, 인간의 참된 자유 실현 |

## 10. 글의 특징

| 특징 | 내용 |
|------|------|
| **이론적 성격** | 특정 작가/작품 비평이 아닌 문학 해석학 일반론 |
| **철학적 토대** | 딜타이 해석학, 실존주의, 마르크스주의 변증법의 종합 |
| **발생론적 접근** | 독서 현상에서 출발하여 해석의 전제 조건 탐구 |
| **주체 중심성** | 인간의 실천적 주체성을 문학 해석의 핵심으로 설정 |
| **변증법적 구조** | 개체와 전체, 자유와 필연, 부분과 전체의 상호 관계 |
