- **저자**: 김우창

- **성격**: 철학 해설, 비판 이론 소개

## 3. 하버마스의 주요 저서

| 저서 | 내용 |
|------|------|
| **『인식과 관심(Erkenntnis und Interesse)』** | 학문의 인식론, 모든 지식에 인간적 관심이 선험적 구성의 바탕 |
| **『이론과 실천(Theorie und Praxis)』** | 이론의 실천적 의미, 홉스·마키아벨리·모어 비판 |
| **『후기 자본주의에 있어서의 정당성의 문제』** | 전통적 규범 해체 후 자유로운 언어 공간으로의 이행 |

## 4. 핵심 개념어

| 개념 | 설명 |
|------|------|
| **비판 이론(Critical Theory)** | 프랑크푸르트 학파의 사회 비판 이론 |
| **도구적 이성** | 특수한 이해관계에 봉사하는 왜곡된 이성 |
| **과학주의적 이성** | 역사적 구성을 등한시하고 현상을 초역사적 실재로 긍정 |
| **실천적 관심** | 학문의 구극적 근거, 해방에의 관심 |
| **이상적 언어 상황** | 지배 관계에서 해방되어 자유로이 대화할 수 있는 상황 |
| **간주관적(intersubjective) 구조** | 인간 존재의 기본 구조, 주관과 객관을 넘어섬 |
| **공적 공간** | 아렌트, 행동과 언어가 교환되는 공간 |
| **공적 행복** | 아렌트, 공적 공간에서의 행동이 주는 행복 |

## 5. 글의 논지 구조

```
1. 서론: 하버마스의 특징
   └── 다방면적 박학
   └── 철학, 사회학, 정치학, 언어 심리학의 종합
   └── 리히트하임: 헤겔에 비교
   └── 이론과 실천의 관계가 핵심

2. 프랑크푸르트 학파의 배경
   └── 현대 선진 자본주의 사회 비판
   └── 마르크스의 자본주의 비판 계승
   └── 경제 관계 → 이데올로기적 조작 과정
   └── 합리성/이성 비판이 중심

3. 이성의 왜곡
   └── 실증주의적 합리성의 문제
   └── 역사적 구성에 대한 반성 등한시
   └── 도구적 이성 = 억압의 수단
   └── 호르크하이머, 아도르노, 마르쿠제

4. 이성의 참다운 모습
   └── 초월의 원리, 부정의 원리
   └── 비판적 기능
   └── 인간의 전면적 해방 지향
   └── 헤겔의 변증법 + 정신 분석

5. 하버마스의 이성 비판
   └── 실천적 차원의 회복
   └── 기술적 이성 → 실천적 관심 → 해방의 관심
   └── 『인식과 관심』: 관심의 선험적 구성 작용

6. 실천의 계기로서의 이론
   └── 객관적 여건 + 이론적 반성
   └── 바른 이론 = 실천의 한 요인
   └── 통속적 마르크시즘의 객관주의 비판

7. 이상적 언어 상황
   └── 모든 이론과 실천의 구극적 이념
   └── 지배 관계에서 해방된 대화
   └── 진리의 문제에도 중요한 의미
   └── 규범 타당성의 이론적 해결

8. 퍼스와 하버마스
   └── 퍼스: 과학적 합리성 = 과학자 공동체
   └── 하버마스: 이성의 근거 = 자유로운 언어 공간

9. 간주관적 구조
   └── 민주적 휴머니즘에 대한 신념
   └── 인간 존재의 간주관적 구조
   └── 주관/객관 = 참다운 객관성과 거리

10. 법칙적 관심 vs 행동
    └── 법칙적 관심 = 통제, 조작
    └── 행동 = 사회적 공간에서의 상호 작용
    └── 예나 시절의 헤겔: 노동 + 상호 작용

11. 한나 아렌트와의 관계
    └── 공적 공간에서의 행동과 언어
    └── 인간 본질의 표현, 공적 행복
    └── 하버마스: 사회 개혁 실천 강조
```

## 6. 프랑크푸르트 학파의 이성 비판

### 6.1 이성의 왜곡 형태

| 형태 | 내용 |
|------|------|
| **과학주의적 이성** | 역사적 구성에 대한 선험적 반성 등한시 |
| **도구적 이성** | 특수한 이해관계의 목적에 봉사 |
| **실증주의적 합리성** | 현상을 초역사적 실재로 긍정 |

### 6.2 이성의 참다운 모습

| 특징 | 내용 |
|------|------|
| **초월의 원리** | 현실 속에 있으면서 현실을 넘어섬 |
| **부정의 원리** | 끊임없는 비판적 기능 |
| **해방 지향** | 인간의 전면적 해방 |

## 7. 하버마스의 핵심 주장

| 주장 | 내용 |
|------|------|
| 학문의 근거 | "학문의 구극적인 근거는 '실천적인 관심'에 있으며, 다시 '해방에의 관심'에 의하여 규명되어야 한다" |
| 관심의 구성 작용 | "모든 지식에서 인간적인 관심은 [...] 학문의 대상과 방향에 대하여 선험적 구성의 바탕이 된다" |
| 가치 중립성 비판 | "소위 '가치 중립적'인 사회 과학 또는 인간 과학은 스스로의 입각지를 모르고 있는 데에서 스스로의 객관 타당성을 믿게 되고 구극적으로는 반성의 저쪽에 있는 억압의 지평에 의하여 통제당하게 되는 것이다" |
| 이론과 실천 | "바른 이론은 실천의 한 요인, 나아가서 실천 그 자체이기까지 한 것이다" |
| 이상적 언어 상황 | "모든 이론과 실천의 구극적인 이념을 '이상적 언어 상황'에 집약한다. [...] 어떤 상황에 참여하는 모든 사람이 지배, 피지배 관계로부터 해방되어 자유로이 대화를 교환할 수 있는 상황" |

## 8. 마르크스 비판

| 비판 내용 | 근거 |
|-----------|------|

| **대안** | 예나 시절 헤겔: 노동 + 주체적 인간의 상호 작용 (두 개의 독립적 변수) |

## 9. 법칙적 관심 vs 행동

| 구분 | 법칙적 관심 | 행동 |
|------|-------------|------|
| **목표** | 통제, 조작 | 상호 작용 |
| **주체** | 한 사람의 사변과 폭력적 의지로 가능 | 다수의 주체적 인간 필요 |
| **공간** | 인간 다수성과 무관 | 사회적 공간 |
| **예** | 홉스, 마키아벨리, 토머스 모어 | 공적 공간에서의 대화 |

## 12. 핵심 논지 요약

1. **프랑크푸르트 학파**: 현대 자본주의 사회의 이데올로기적 조작 비판
2. **이성의 왜곡**: 도구적 이성, 과학주의적 이성이 억압의 수단이 됨
3. **이성의 회복**: 실천적 관심, 해방에의 관심을 통한 이성의 회복
4. **이론과 실천**: 바른 이론은 실천 그 자체, 객관적 여건 + 이론적 반성
5. **이상적 언어 상황**: 지배 관계에서 해방된 자유로운 대화의 공간
6. **간주관적 구조**: 인간 존재의 기본 구조, 민주적 휴머니즘
7. **행동의 중요성**: 법칙적 통제가 아닌 사회적 공간에서의 상호 작용
