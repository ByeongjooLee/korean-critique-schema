# 김우창 비평의 지적 지형도

## 개요
본 문서는 김우창의 비평 22편을 분석하여 그가 인용한 서양 사상가/철학자와 분석 대상 한국 작가를 정리한 것이다.

## 4. 김우창 비평의 지적 특성

### 4.1 핵심 이론적 축

```
1. 데카르트적 주체론
   └── 근대적 자아의 형성과 문제
   └── 코기토(cogito)의 문학적 의미
   └── 한국 근대문학과 근대적 주체 의식

2. 현상학적 방법
   └── 하이데거: 존재론적 접근
   └── 메를로퐁티: 신체성, 지각의 현상학
   └── 사르트르: 상황과 주체

3. 비판이론
   └── 프랑크푸르트 학파
   └── 루카치: 소설의 이론, 물화
   └── 골드만: 발생론적 구조주의

4. 영미 신비평/모더니즘
   └── T.S. 엘리엇: 객관적 상관물, 형이상시
   └── 발레리: 순수시, 언어의 자율성
```

### 4.2 이론 활용 패턴

| 분석 대상 | 주요 활용 이론 |
|-----------|----------------|
| **시 분석** | 엘리엇, 발레리, 하이데거, 키르케고르 |
| **소설 분석** | 루카치, 골드만, 데카르트, 이안 와트 |
| **비평가론** | 엘리엇, 레비스트로스, 마르크스 |
| **이론적 에세이** | 하버마스, 사르트르, 메를로퐁티, 소쉬르 |
| **교육/사회 에세이** | 듀이, 쿤, 마르쿠제, 아렌트 |

### 4.3 주요 개념어 네트워크

```
주체(subject) ─── 데카르트 ─── 코기토
      │
      ├── 상황(situation) ─── 사르트르
      │
      ├── 존재(Being) ─── 하이데거
      │
      └── 간주관성(intersubjectivity) ─── 메를로퐁티, 하버마스

소설 ─── 루카치 ─── 총체성
  │
  ├── 골드만 ─── 세계관/집단의식
  │
  └── 이안 와트 ─── 형식적 리얼리즘

시 ─── 엘리엇 ─── 객관적 상관물
  │
  ├── 발레리 ─── 순수시
  │
  └── 하이데거 ─── 언어의 존재론
```

## 5. 결론: 김우창 비평의 지적 좌표

### 동양과 서양 사이
- 서양 이론의 적극적 수용과 비판적 전유
- 한국 문학의 특수성에 대한 천착

### 전통과 근대 사이
- 데카르트적 근대성의 비판적 검토
- 전통적 세계관과의 대화

### 개인과 사회 사이
- 주체성 문제에 대한 지속적 탐구
- 사회적 맥락에 대한 관심 (루카치, 골드만, 하버마스)

### 문학과 철학 사이
- 문학 비평에 철학적 깊이 부여

## 부록: 인용 빈도 통계

### 전체 인용 사상가 수: 약 60명

### 분야별 비율
- 철학: 35%
- 문학이론/비평: 25%
- 비판이론/마르크스주의: 20%
- 사회학/정치철학: 10%
- 심리학/정신분석: 5%
- 기타: 5%

### 시대별 비율
- 고대/중세: 5% (플라톤, 아우구스티누스)
- 근대(17-19세기): 25% (데카르트, 파스칼, 헤겔, 키르케고르)
- 현대(20세기): 70%
