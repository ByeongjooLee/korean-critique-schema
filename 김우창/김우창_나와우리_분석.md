## 2. 인용 예시문

### 2.1 토머스 홉스 (Thomas Hobbes)

> "**토머스 홉스(Thomas Hobbes)**가, 있을 수 있는 삶의 한 형태로서 자연의 상태란 것을 상정하고 그것을 모든 사람에 대한 모든 사람의 싸움으로 설명한 것은 유명한 일이다. 그는, 이 자연의 상태에서는 문화의 기예(技藝)가 위축되고 사람들의 마음은 '끊임없는 공포감, 비명횡사(非命橫死)의 가능성'에 떨고, 삶은 '외롭고 초라하고 야비하고 짐승스러우며 짧은 것'이 된다고 말하였다."

### 2.2 윌리엄 제임스 (William James)

> "**윌리엄 제임스**가 그의 『심리학(心理學)의 원리(原理)』에서 경험적인 관점에서 파악한 자아(自我, self)를 설명하면서 든 예를 빌려, 자아란 '내 신체, 정신 능력, 의지, 집, 처자(妻子), 내가 가지고 있는 토지, 말(馬), 요트, 은행, 예금 통장', 이 모든 것을 포함한 '내 것이라고 부를 수 있는 일체'라고 말해 버릴 수만은 없다."

### 2.3 피에르 카우프만 (Pierre Kaufmann)

> "사람의 공간 경험에 대하여 현상학적 분석을 시도한 프랑스의 철학자 **피에르 카우프만(Pierre Kaufmann)**은 어떤 물건이 물건으로서 지각되는 '감각적 의미의 장(場)'은 타자(他者)와의 '공존(共存)의 장(場)'이라고 말한다."

### 2.4 조지 허버트 미드 (George Herbert Mead)

> "'일반화된 타자'는 **조지 허버트 미드(George Herbert Mead)**로부터 빌려 온 것이지만, 우리가 지금껏 주로 사용해 온 원근법의 비유에 관련하여, 발생적으로 우리의 자아 활동이 타자의 존재에 결부되어 있는가의 보기는 메를로퐁티로부터 더 쉽게 끌어올 수 있다."

### 2.5 메를로퐁티 (Merleau-Ponty)

> "콜레주 드 프랑스의 아동 심리학 교수로서 그가 행한 강의 초안, 「어린이에 있어서의 다른 사람과의 관계」에서, 그는 유아(幼兒)의 지적 성장에 관한 여러 실험적 연구를 종합하여 거기에 철학적인 분석을 가하고 있는데 [...] **메를로퐁티**에 의하면 어린이에 있어서의 자아의 발달은 성장의 시간적 단계에서 비교적 나중에 속하며 처음에는 자타(自他)를 구분하지 않는 혼돈의 상태가 있을 뿐이라고 한다."

### 2.6 하이데거 (Heidegger)

> "**하이데거**는 존재로부터의 이탈을 극복하는 방법은 존재의 부재(不在)가 곧 존재의 한 양상이라는 것을 깨닫는 것이라고 말한 바 있다."

### 2.7 한나 아렌트 (Hannah Arendt)

> "여러 가지 약점에도 불구하고 **한나 아렌트(Hannah Arendt)**는 공동체적 공간으로서의 정치의 장(場)에 대하여 가장 독창적이고 심오한 해석을 가한 정치 철학자 중의 한 사람이다. 이 공간이 사람과 사람의 약속으로 성립하고(또 용서를 통하여 이 약속은 해제될 수 있다.) 그것이 근본에 있어서 '뛰어남(excellence)'을 자랑함으로써 개체적인 실존에 공존적인 보강(補强)을 얻을 수 있게 하는 '보임의 공간'이라는 생각은 아렌트의 저서에 도처에 보인다."

## 3. 핵심 개념어

| 개념 | 설명 |
|------|------|
| **만인전쟁(萬人戰爭)** | 홉스의 자연 상태, 모든 사람의 모든 사람에 대한 싸움 |
| **자아(自我, ego/self)** | 세계에로 열려 있는 관점, 움직이는 점 |
| **원근법** | 여러 관점의 종합화, 타자의 관점을 포함 |
| **일반화된 타자** | 미드의 개념, 자아 발생과 성장에 대칭적으로 발전하는 타자 |
| **거울의 경험** | 메를로퐁티, 자아의식의 시작, 타자의 관점 취함 |
| **공존의 장(場)** | 카우프만, 감각적 의미의 장 = 타자와의 공존의 장 |
| **주체와 객체의 변증법** | 주체성의 객체화와 객체의 주체화 |
| **스타일** | 세계에 열려 있는 인간의 존재 방식, 개성의 표현 |
| **약속과 현양(顯揚)의 공간** | 아렌트, 정치적 공간의 본질 |

## 4. 글의 논지 구조

```
1. 서론: 만인전쟁
   └── 홉스의 자연 상태
   └── 한국 사회의 현실
   └── 만인전쟁의 본질: 이용하고 이용당하는 관계
   └── 해결책의 모색: 절대 군주제, 도덕적 질서

2. 개인과 사회
   └── 의지의 상충과 사회 전체성
   └── 강제된 전체성의 문제
   └── 문학에서의 개인과 전체 긴장
   └── 순수 문학 vs 참여 문학

3. 자아의 구조
   └── 윌리엄 제임스: 경험적 자아
   └── 자아 = 세계에로의 열림의 한 방식
   └── 데카르트: "나는 생각한다, 고로 존재한다"
   └── 관점의 선택과 역할(role)

4. 원근법과 타자
   └── 원근법 = 여러 관점의 종합화
   └── 3차원 지각 = 관점의 이동
   └── 카우프만: 감각적 의미의 장 = 공존의 장
   └── 타자 = 의미의 공동수립자

5. 자아의 발생과 타자
   └── 미드: 일반화된 타자
   └── 메를로퐁티: 거울의 경험
   └── 자타를 구분하지 않는 혼돈 상태
   └── 공존의 통일 작용

6. 자아, 전체, 문학
   └── 문학 = '나'와 다른 '나'의 교감
   └── 공동 근거에의 복귀 호소
   └── 왜곡된 공존 관계의 인식
   └── 가능성으로서의 타락

7. 주체와 객체의 변증법
   └── 원초적 '우리' = 움직임의 체계
   └── 부정적 공존 관계의 객체적 성격
   └── 객체의 내면화와 주체에로의 전성
   └── 운명의 자각 = 살풀이의 시작

8. 주체성의 회복
   └── 개인적 체험에서 출발
   └── 삶의 조건의 지평적 삼투
   └── 한계의 자각
   └── 객체적 조건의 역사적 형성 가능성

9. 스타일과 개성
   └── 추사의 글씨: 독특한 존재 방식
   └── 스타일 = 세계와의 교섭
   └── 개인적 스타일 → 시대의 스타일
   └── 언어와의 교호 작용

10. 결론: 약속과 현양의 공간
    └── 정치적 행위 = 공동 공간 마련
    └── 사회 = 사랑의 공간
    └── 약속으로서의 긴장
    └── 한나 아렌트: 뛰어남의 현양
```

## 5. 자아론 분석

### 5.1 자아의 정의

| 관점 | 내용 |
|------|------|
| **경험주의적** | 윌리엄 제임스: 신체, 재산 등 "내 것이라고 부를 수 있는 일체" |
| **존재론적** | 세계에로 열려 있는 몸뚱이, 열림의 한 방식 |
| **기하학적** | 원근법 속에 있는 움직이는 점, 관점의 선택 가능 |
| **사회적** | 타자와의 공존 관계 속에서 형성 |

### 5.2 자아와 타자의 관계

| 단계 | 내용 |
|------|------|
| **원초적 상태** | 자타를 구분하지 않는 혼돈(메를로퐁티) |
| **거울의 경험** | 타자의 관점 취함으로써 자아의식 시작 |
| **지능의 발달** | 관점의 다변적 보완성에 대한 인식 능력 |
| **성인의 상태** | '나와 남'의 상보 관계, 평형 상태 |

## 6. 주체와 객체의 변증법

| 단계 | 내용 |
|------|------|
| **1단계** | 개인적 체험만이 주어진 것 |
| **2단계** | 외부적 조건(계급, 민족)은 지평으로만 삼투 |
| **3단계** | 삶의 주체적 소유 노력 → 한계에 도달 |
| **4단계** | 객체적 조건의 내면화 자각 (운명의 자각) |
| **5단계** | 대상적 파악 → 소외 또는 새로운 변화의 전기 |
| **6단계** | 역사적 형성의 가능성으로서 '우리' 회복 |

## 7. 문학론적 명제

| 명제 | 내용 |
|------|------|
| 문학의 정의 | "문학은 삶의 공통적 근거를 통하여 이루어지는 '나'와 다른 '나'와의 교감이다" |
| 문학의 기능 | "예술 특히 문학의 한 기능은 한 사회가 추상적인 가치 개념으로 고착시키는 그 사회의 전체성을 구체적인 개인적 실존과의 함수 관계에서 검토하는 일" |
| 경험과 가능성 | "시인은 경험의 입장을 벗어날 수 없으나 역설적으로 경험을 가능성의 지평 속에 파악함으로써 경험 그 자체가 지닌 미래에로의 초월성을 보여 줄 수 있다" |
| 스타일의 의미 | "개성적인 스타일은 세계에 열려 있는 한 인간의 존재 방식이다. 이것은 그 사람에게 유니크한 것이면서 또 세계와의 교섭 없이는 성립되지 아니한다" |
| 언어의 창조 | "나의 언어는 타자의 언어와의 교호작용을 통한 자기 점검이다. 시적 언어에서 언어는 가장 개성적이면서 또 세계 그것의 요구에 화답하는 것이다" |

## 8. 정치철학적 명제

| 명제 | 내용 |
|------|------|
| 만인전쟁의 본질 | "사람과 사람의 관계가 이용하고 이용당하는 관계로서만 성립한다" |
| 강제된 전체성 | "개인의 동의 없는 질서의 강제는 표면적인 평화에도 불구하고 만인전쟁의 연장에 불과" |
| 사회의 의미 | "사회는 마땅히 사랑의 공간으로 성립해야 한다. 우리는 거기에서 생존의 필연을 사람 사이의 약속으로서 바꾸어 놓을 수 있다" |
| 정치의 공간 | "이상적인 사회는 이것[개인의 형이상학적 가능성]을 최대한으로 현양(顯揚)할 수 있는 공간으로서 성립한다" (아렌트) |

## 11. 핵심 논지 요약

1. **만인전쟁의 현실**: 한국 사회는 홉스적 만인전쟁 상태
2. **강제된 전체성의 한계**: 외부적 강제는 진정한 평화가 아님
3. **자아의 사회적 구성**: 자아는 타자와의 공존 관계 속에서 형성
4. **원근법과 타자**: 우리의 지각 자체가 타자를 포함
5. **주체-객체 변증법**: 객체의 내면화 자각을 통한 해방
6. **문학의 역할**: 공동 근거에의 복귀 호소, 가능성의 지평 제시
7. **사회의 이상**: 약속과 현양의 공간, 사랑의 공간
