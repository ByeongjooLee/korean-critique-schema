## 5. 김우창의 평가 분석 (긍정/부정/중립)

### 5.1 긍정적 평가

| 평가 내용 | 근거 |
|-----------|------|
| 기획의 의의 | "19세기 후반으로부터 해방까지의 역사를 소설로 옮겨 본다는 것은 그 기도(企圖)에 있어서 벌써 의의 있는 일" |
| 민족 주체성 논의에 대한 기여 | "이 주체성의 논의에서 그 살아 있는 의미를... 이 소설만큼 설득력 있게 이야기하고 있는 경우도 찾아보기 어렵지 않을까" |
| 테마의 설득력 | "부분적으로 기복이 있으면서도『북간도』의 테마는 설득력 있게 독자에게 전달된다" |
| 역사적 사건의 재생 | "대사건들이 간결하면서도 실감 있게 재생되어" |
| 대사건과 주인공 생활의 편집 | "대사건과 주인공들의 생활과의 교호 작용(交互作用)은 능숙한 솜씨로 편집 정리되어 있다" |
| 중요한 업적 | "오늘날 우리 문학이 처해 있는 단계에서『북간도』는 하나의 중요 업적" |
| 기록과 증언의 가치 | "이것은 중요한 기록(記錄)이며 중요한 증언(證言)이다" |
| 진지한 노력 | "이 기록과 증언은 진지하고 심각한 노력 속에 이루어진 것" |

### 5.2 부정적 평가

| 평가 내용 | 근거 |
|-----------|------|
| 내적 주제의 변증법 부재 인상 | "얼른 보아 이 소설은 뚜렷한 내적인 주제의 변증법을 가지고 있지 않다는 인상을 주기 쉽다" |
| 인물 묘사의 희미함 | "『북간도』의 인물이 극히 희미한 정도로밖에 그려져 있지 않다" |
| 스타일의 특징 부재 | "그 스타일이 기껏해야 특징 없이 담담하다" |
| 소도구의 빈약함 | "『북간도』의 세계가, 독자적인 현실성을 갖기에는, 현실의 소도구(小道具)에 너무나 빈약하다" |
| 낮은 예술적 압력 | "이 작품이 전체적으로 매우 낮은 예술적 압력밖에 갖지 못했다" |
| 강력한 인상의 부재 | "독자의 마음에 뚜렷하고 강력한 인상을 새겨 놓지 못한다" |
| 갈등 주동 세력에 대한 이해 부족 | "갈등의 주동 세력에 대한 이해가 올바르게 되어 있지 않기 때문" |
| 정면 대결 회피 | "작자가 이 세력들과 정면으로 대결하기를 회피하고 있기 때문" |
| 감자밭 사건의 조작성 | "이 감자밭 사건 같은 것은 순전히 조작이라고 할 수밖에 없다" |
| 사이비 상징주의 수법 | "안수길 씨는 사이비 상징주의 수법을 짚고서 핵심적인 갈등을 정면으로 극화하는 고역(苦役)을 피하려는 경향" |
| 갈등의 첨예화 부재 | "이 소설에서 어떠한 갈등도 첨예화되지 않음" |
| 윤리적·철학적 차원의 결여 | "윤리적·철학적 차원의 결여는 이 소설의 보편적인 가치를 현저하게 떨어뜨린다" |
| 등장인물의 수동성 | "이한복에서 정수에 이르는 주인공들이 저항적인 인물이기는 하지만 그들의 저항은 결코 적극적인 것이 되는 법이 없다" |
| 감동적 인간드라마의 부재 | "이 작품에 있어서의 갈등의 둔화는 등장인물들의 수동성에도 관계된다... 참으로 감동적인 인간드라마가 없다" |
| 정수 자수의 문제 | "정수의 자수는 불필요한 삽화인 것 같다... 지금까지 전개되어 온 소설의 테제를 부정해 버리는 결과" |

### 5.3 중립적 평가 (분석적 서술)

| 평가 내용 | 근거 |
|-----------|------|
| 저항과 적응의 두 반응 양식 | 이한복(저항) vs 장치덕(적응)의 대비적 분석 |
| 민족 주체성의 전개 과정 | "처음에 제 것을 지키고자 하는 인간 본연의 충동... 점점 강력하게, 이것은 하나의 필수 불가결한 삶의 조건으로 파악" |
| 자치의 이상 | "자치(自治)의 이상은 이 소설의 목가(牧歌)적인 핵심" |
| 힘의 주제 | "힘의 결여에 말미암은 것" - 사포대 조직과 무장화 |

### 5.4 종합 평가

김우창의『북간도』평가는 **공과(功過)를 명확히 구분하는 균형 잡힌 비평**이다:

1. **주제의 중요성 인정**: 민족 주체성이라는 주제의 역사적·현실적 의의를 높이 평가한다.

2. **기법적 한계의 지적**: 인물 묘사, 갈등의 첨예화, 상징 수법 등 소설 기술의 문제점을 구체적으로 지적한다.

3. **구조적 문제 분석**: 정수의 자수가 작품 테제를 훼손한다는 구조적 비판을 제기한다.

4. **철학적 깊이의 요구**: 윤리적·철학적 차원의 결여가 보편적 가치를 떨어뜨린다고 비판한다.

5. **역사적 의의 인정**: 비판에도 불구하고 "중요한 기록이며 중요한 증언"으로서의 가치를 인정한다.

**결론**: "이러한 험담은 보다 완벽한 소설을 기대하는 우리의 희망적인 선의(善意)를 나타낸 것에 불과하다"는 표현처럼, 비판은 작품의 가치를 부정하기 위한 것이 아니라 더 완벽한 문학을 기대하는 건설적 비평이다.

## 7. 글의 논지 구조

```
1. 서론: 『북간도』의 성격
   └── 북간도 한국 이민의 연대기
   └── 동북아 국제 정치의 역학 관계
   └── 민족 주체성의 문제

2. 주제의 전개: 4대에 걸친 가족사
   └── 제1대 이한복: 생존권의 주장, 저항
   └── 제3대 창윤: 자치의 이상
   └── 제4대 정수: 독립군 활동, 자수

3. 저항과 적응의 대비
   └── 저항: 이한복의 상투 유지
   └── 적응: 장치덕의 삭발, 귀화
   └── 적응의 실패: 최삼봉, 노덕심, 이기형

4. 외부 세력의 변화
   └── 청국의 압력: 흑복 변발, 입적 귀화
   └── 일본의 이중성: 보호와 이용
   └── 역사적 대사건들의 소설화

5. 주체성 확립의 필요성
   └── 주체적 법질서 없이는 최소한의 안전도 불가
   └── 힘/실력의 주제
   └── 자치의 이상

6. 비판적 검토
   └── 예술적 압력의 부족
   └── 갈등의 첨예화 실패
   └── 사이비 상징주의 수법
   └── 등장인물의 수동성
   └── 정수 자수의 문제

7. 결론: 업적의 인정
   └── 중요한 기록과 증언
   └── 진지하고 심각한 노력의 산물
```
