## 4. 김우창의 평가 분석 (긍정/부정/중립)

### 4.1 긍정적 평가

| 평가 내용 | 근거 |
|-----------|------|
| 시적 언어의 독창성 | "미당 선생의 언어처럼 시적인 언어도 드문 것... 우리 일상생활의 무엇이든지 그대로 시가 되어 버린다" |
| 시적인 것과 일상적인 것의 혼연일체 | "시적인 것과 일상적인 것이 혼연일체를 이룬다" |
| 장인의 솜씨 | "매체(媒體)를 손끝에 익히고 익혀 완전히 자신의 일부를 만들어 버린 장인(匠人)의 솜씨" |
| 토속적 세계의 탐구 | "다른 어떤 현대 시인의 시보다도 토속적 세계에 깊이 뿌리를 내리고 있다" |
| 샤머니즘의 시적 착반 | "샤머니즘과 같은 원시적인 세계를 착반하고... 최초의 주요한 시인" |
| 역사적 통찰 | "어떤 단순화된 정치 논설보다도 해방을 그 근원적인 의미에서 파악" |
| 현실 비판의 날카로움 | "직관적 현실 비판은 사뭇 날카로운 것이다" |
| 체험적 진실의 무게 | "미당 선생의 시가 담고 있는 체험적인 진실의 무게를 깊이 느끼지 않을 수 없다" |
| 한국적 체험의 현대적 표현 | "한의 세계에서 자라나오고 이 세계에 현대적인 표현을 주었다" |

### 4.2 부정적 평가 (한계 지적)

| 평가 내용 | 근거 |
|-----------|------|
| 정치시의 상투성 | "정치 상황에 대한 파악과 비판은... 대체로 상투적인 틀을 벗어나지 못하고 있는 것이어서 반드시 성공적이라고 할 수는 없지만" |
| 이존책의 한계 | "이존(以存)의 방책만을 유일한 인간의 현실로 받아들일 수 없는 상태에 이른 것이 아닌가" |
| 새로운 단계의 요구 | "구부러진 자세로 참고 견디지 않아도 될 상태를 생각할 수 있는 그런 마음의 근본적인 현실을 가져야 할 단계" |

### 4.3 중립적 평가 (분석적 서술)

| 평가 내용 | 근거 |
|-----------|------|
| 자연관의 특성 | "자연을 단순히 평화와 위로의 출처로 생각하지 않고 욕망의 과정으로 본다" |
| 욕망의 형이상학 | "욕망이 이루어지지 않더라도 그 자체로서 스스로의 보람이 된다" |
| 고통과 긍정의 변증법 | "감당하기에 너무나 커다란 고통의 세계에서 시인이 갖는 위안의 필요에서 나온 믿음" |
| 굽음의 철학 | "곧장 가자 하면 갈 수 없는 벼랑길도/굽어서 돌아가기면 갈 수 있는 이치" |
| 한(恨)의 미학 | "전통적·심미적 경험의 기초... 좌절된 욕망의 공간" |

### 4.4 종합 평가

김우창의 서정주 평가는 **깊은 공감과 역사적 맥락화, 그리고 시대적 한계 지적을 결합한 비평**이다:

1. **언어와 세계의 독창성**: 시적인 것과 일상적인 것의 혼연일체, 토속적 세계의 시적 착반을 높이 평가한다.

2. **형이상학적 체계 분석**: 범욕주의, 자연과 인간의 정적 유대, 욕망의 변증법 등 서정주 시의 철학적 기반을 체계적으로 분석한다.

3. **역사적 체험의 표현**: 한(恨)의 미학과 이존책(以存策)이 한국인의 역사적 체험을 반영함을 인정한다.

4. **시대적 한계 제기**: 1970년대 후반에는 "구부러진 자세로 참고 견디지 않아도 될 상태"를 생각해야 한다며 새로운 방향을 요청한다.

5. **미래적 기대**: "오늘날의 현실과 보다 나을 수 있는 미래의 현실에 대해서도... 시적인 예지를 보여 주길" 희망한다.

## 6. 글의 논지 구조

```
1. 서론: 시적 세계와 여느 세계
   └── 전통적 시의 세계: 산, 구름, 달, 꽃
   └── 미당의 새로운 경지: 시적인 것과 일상적인 것의 혼연일체
   └── 언어의 특징: 장인의 솜씨

2. 「우중유제」 분석
   └── 성행위와 홍시의 비유
   └── 몰입과 초연의 논리
   └── 생명 충동과 종족 보존의 리듬
   └── 신라 시대와 오늘의 대비

3. 토속적 세계의 탐구
   └── 샤머니즘적 세계관
   └── 자연 속 인간의 원초적 이미지
   └── 단군 신화, 곰과 범의 상징

4. 자연과 인간의 정적 유대
   └── 범욕주의: 자연도 욕망의 표현
   └── 인간의 욕망과 자연의 동질성
   └── 역사도 자연의 충동에 의해 연결

5. 욕망의 형이상학
   └── 욕망의 좌절과 변형
   └── 동정적 마술(sympathetic magic)
   └── 리비도의 변형, 보상, 승화
   └── 무당: 억압된 충동의 대표자

6. 고통과 구원
   └── 고통을 통한 깊이
   └── 수난의 민족상
   └── 욕망 자체의 보람

7. 불행한 세계와 위안
   └── 생존의 불안감
   └── 고통 속 긍정의 교훈
   └── 「뻔디기」: 사회의식으로의 확장

8. 굽음의 철학
   └── 완곡의 실천 철학
   └── 이존책(以存策)
   └── 「복 받을 처녀」의 현실주의

9. 한(恨)의 미학
   └── 좌절된 욕망의 공간
   └── 숙명과 유대의 변증법
   └── 한국적 체험의 현대적 표현

10. 결론: 새로운 단계의 요청
    └── 이존책의 한계
    └── 구부러지지 않아도 될 상태의 희망
```

## 7. 서정주 시의 특징 정리

| 특징 | 내용 |
|------|------|
| **언어** | 시적 언어와 일상 언어의 혼연일체, 토속어·사투리의 시적 효과 |
| **소재** | 전통적 시적 물건 + 일상의 물건(매화틀, 놋요강, 건건이, 뻔디기 등) |
| **세계관** | 샤머니즘, 범욕주의, 자연과 인간의 정적 유대 |
| **역사관** | 자연의 충동이 역사의 근본 동력, 시대 내림세에 대한 비판 |
| **실천철학** | 굽음의 철학, 완곡한 방법, 이존책 |
| **감정** | 한(恨), 그리움, 욕망의 좌절과 승화 |

## 8. 「우중유제」 분석 요약

| 분석 층위 | 내용 |
|-----------|------|
| **표면적 의미** | 성행위 중 홍시에 마음이 쏠리는 상황 |

| **역사적 대비** | 신라(풍요) vs 오늘(궁핍) |
| **언어적 특징** | '사내', '진땀', '계집', '그 짓' - 경멸적 거리감 |

## 9. 범욕주의 세계관의 구조

```
자연 = 욕망/그리움의 표현
    ↓
인간 = 자연과 동질적 충동
    ↓
역사 = 자연의 충동에 의한 연결
    ↓
욕망의 좌절 = 한(恨)
    ↓
간접적 실현 = 동정적 마술, 변형, 승화
    ↓
굽음의 철학 = 이존책
```

## 10. 한(恨)의 미학 도식

| 요소 | 내용 |
|------|------|
| **세계** | 욕망의 장(場) |
| **욕망의 상태** | 이루어지기보다 좌절 |
| **결과** | 좌절된 욕망의 공간 = 한의 공간 |
| **태도** | 숙명으로서의 한 → 세상을 쉽게 받아들임 |
| **가능성** | 정적(情的) 유대 → 한 해소의 가능성 |
| **사회적 기반** | 정적 관계에 얽힌 신분사회의 구석 |
